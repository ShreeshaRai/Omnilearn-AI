OmniLearn AI – Intelligent Multimodal Learning Assistant

OmniLearn AI is a full-stack AI-powered learning platform that combines chat-based learning, quiz generation, voice interaction, and image analysis into one unified system.

This project demonstrates the integration of modern AI capabilities with a real-world web application.

🧠 Features
	•	💬 AI Chat Assistant
Ask any question and receive intelligent responses in real-time
	•	🎤 Voice Input
Speak instead of typing using speech recognition
	•	🔊 Text-to-Speech
AI responses are converted into voice output
	•	📷 Image Capture & Analysis
Capture images using camera and process them
	•	📝 AI Quiz Generator
Generate quizzes instantly on any topic
	•	📊 History Tracking
View previous interactions and learning progress
